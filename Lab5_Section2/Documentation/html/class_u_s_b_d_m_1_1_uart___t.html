Detailed Description

template<class Info>
class USBDM::Uart_T< Info >

Template class representing an UART interface.

Example

// Instantiate interface
Uart *uart0 = new USBDM::Uart_T<Uart0Info>(115200);
for(int i=0; i++;) {
uart<<"Hello world, i="<<i<<"\n";
}
Template Parameters
InfoClass describing UART hardware
