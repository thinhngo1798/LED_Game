enum USBDM::SimAdc0TriggerMode

Selects the ADC0 trigger mode.

_Pdb - ADC is triggered by PDB which selects the pretrigger (SC1[0..n]/R[0..n] _Alt_PreTrigger_0 - ADC is triggered by SimAdc0Trigger selection and uses pretrigger 0 = A (SC1[0]/R[0]) _Alt_PreTrigger_1 - ADC is triggered by SimAdc0Trigger selection and uses pretrigger 1 = B (SC1[1]/R[1])
