◆ setCallback()

void USBDM::Endpoint::setCallback ( EndpointState(*)(EndpointState)  callback)
inlineinherited

Set callback to execute on completion of data and handshake phases of transaction.

This callback routine is called with parameter endpointState which is the state of endpoint before completion of transaction:

  • After an IN transaction with endpointState = EPDataIn, EPStatusIn.
  • After an OUT transaction with endpointState = EPDataOut, EPStatusOut. The endpoint state will be set to EPIdle before calling this routine.
Parameters
[in]callbackThe call-back function to execute
May be nullptr to remove callback

◆ fCallback

EndpointState(* USBDM::Endpoint::fCallback) (EndpointState endpointState)
protectedinherited

Callback used on completion of data and handshake phases of transaction.

For a control endpoint this routine is called:

  • After an IN transaction with endpointState = EPStatusIn.
  • After an OUT transaction with endpointState = EPStatusOut.

For other types of endpoint this routine is called:

  • After an IN transaction with endpointState = EPDataIn.
  • After an OUT transaction with endpointState = EPDataOut.
Parameters
[in]endpointStateState of endpoint before completion of transaction
Returns
The endpoint state to set after call-back
