Detailed Description

template<class Info, const unsigned bitNum, Polarity polarity>
class USBDM::Gpio_T< Info, bitNum, polarity >

Create GPIO from GpioInfo class.

Template Parameters
InfoGpio information class
bitNumBit number within PORT/GPIO
polarityPolarity of pin. Either ActiveHigh or ActiveLow

Set pin as digital I/O.

Pin is initially set as an input. Use SetIn() and SetOut() to change direction. If open-drain then input function may meaningfully be used while set as output

Note
Resets the Pin Control Register value (PCR value).
Resets the pin output value to the inactive state
Parameters
[in]pinPullOne of PinPull_None, PinPull_Up, PinPull_Down
[in]pinDriveStrengthOne of PinDriveStrength_Low, PinDriveStrength_High (defaults to PinDriveLow)
[in]pinDriveModeOne of PinDriveMode_PushPull, PinDriveMode_OpenDrain (defaults to PinPushPull)
[in]pinActionOne of PinAction_None, etc (defaults to PinAction_None)
[in]pinFilterOne of PinFilter_None, PinFilter_Passive (defaults to PinFilter_None)
[in]pinSlewRateOne of PinSlewRate_Slow, PinSlewRate_Fast (defaults to PinSlewRate_Fast)
