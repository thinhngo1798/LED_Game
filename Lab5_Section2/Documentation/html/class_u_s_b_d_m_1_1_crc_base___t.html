Detailed Description

template<class Info>
class USBDM::CrcBase_T< Info >

Template class representing the Cyclic Redundancy Check Monitor.

The cyclic redundancy check (CRC) module generates 16/32-bit CRC code for error detection. The CRC module provides a programmable polynomial, WAS, and other parameters required to implement a 16-bit or 32-bit CRC standard. The 16/32-bit code is calculated for 32 bits of data at a time.

Example CRC methods provided See http://reveng.sourceforge.net/crc-catalogue/

CRC-16/CCIT-FALSE ARC(CRC-16) CRC-32
Width 16 bits 16 bits 32 bits
Polynomial 0x1021 0x8005 0x04C11DB7
Initial seed 0xFFFF 0x0000 0xFFFFFFFF
Input transform None B&B B&B
Output transform None B&B B&B
Complement checksum No No Yes
