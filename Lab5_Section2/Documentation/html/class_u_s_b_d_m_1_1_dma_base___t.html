◆ startSoftwareRequest()

template<class Info >
static void USBDM::DmaBase_T< Info >::startSoftwareRequest ( DmaChannelNum  dmaChannelNum)
inlinestatic

Initiate a DMA software request on a channel.

The channel should be configured beforehand using configureTransfer().
This is a convenience function that allows re-use of already configured channel.
All it does is set the START bit in the existing TCD.
There is no need to use this function for a single request as the START bit may be set
in the original TCD used with configureTransfer().

Parameters
[in]dmaChannelNumChannel being modified
Note
There is no clear option as the flag is automatically cleared by the DMA controller when the transfer starts.
