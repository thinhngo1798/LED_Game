Detailed Description

template<unsigned left, unsigned right, Polarity polarity = ActiveHigh>
class USBDM::GpioBField< left, right, polarity >

Convenience template for GpioB fields.

See Field_T

Usage

using namespace USBDM;
// Instantiate for bit 6 down to 3 of GpioB
using GpioB6_3 = GpioBField<6,3>;
// Set as digital output
GpioB6_3::setOutput();
// Write value to field
GpioB6_3::write(0x53);
// Clear all of field
GpioB6_3::bitClear();
// Clear lower two bits of field
GpioB6_3::bitClear(0x3);
// Set lower two bits of field
GpioB6_3::bitSet(0x3);
// Set as digital input
GpioB6_3::setInput();
// Read pin as int value
int x = GpioB6_3::read();
Template Parameters
leftBit number of leftmost bit in port (inclusive)
rightBit number of rightmost bit in port (inclusive)
polarityPolarity of all pins. Either ActiveHigh or ActiveLow
