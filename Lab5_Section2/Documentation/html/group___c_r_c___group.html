◆ CrcReadComplement

enum USBDM::CrcReadComplement

Complement Read Of CRC Data Register.

Some CRC protocols require the final checksum to be XORed with 0xFFFFFFFF or 0xFFFF. Asserting this enables on the fly complementing of read data.

Enumerator
CrcReadComplement_Inactive 

No complement of CRC data register on read.

CrcReadComplement_Active 

Complement of CRC data register on read.
