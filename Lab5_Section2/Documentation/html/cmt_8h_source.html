117 /**
118  * Template class representing a Carrier Modulator Transmitter.
119  *
120  * Typical modes of operation:
121  * Time mode
122  * Waveform alternates between modulation frequency (mark) and idle (space).
123  * Modulation waveform is controlled by setPrimaryTiming().
124  * Mark/Space time is controlled by setMarkSpaceTiming().
125  * Baseband mode
126  * Waveform alternates between high (mark) and idle (space) without use of modulation.
127  * Mark/Space time is controlled by setMarkSpaceTiming().
128  * FSK mode
129  * Waveform changes between two modulation frequencies(mark) separated by idle time (space) i.e.
130  * (Primary modulation - idle - Secondary modulation - idle) repeated.
