Detailed Description

template<uint32_t clockInfo, uint32_t portAddress, IRQn_Type irqNum, int bitNum, PcrValue defPcrValue>
class USBDM::Pcr_T< clockInfo, portAddress, irqNum, bitNum, defPcrValue >

Template representing a Pin Control Register (PCR)

Code examples:

// Create PCR type
using PortC_3 = USBDM::Pcr_T<SIM_SCGC5_PORTC_MASK, PORTC_BasePtr, 3, USBDM::DEFAULT_PCR>;
// Configure PCR
PortC_3::setPCR(PinPull_Up,PinDriveStrength_High,PinDriveMode_PushPull,PinAction_None,PinFilter_None,PinSlewRate_Fast,PinMux_3);
// Disable Port clock
PortC_3::disableClock();
Template Parameters
clockInfoPCC register address or mask for SIM clock register associated with this PCR
portAddressPORT to be manipulated e.g. PORTA (PCR array)
bitNumBit number e.g. 3
defPcrValueDefault value for PCR (including MUX value)
