Detailed Description

Class to implement simple critical sections by disabling interrupts.

Disables interrupts for a critical section. This would be from the declaration of the object until the end of enclosing block. An object of this class should be declared at the start of a block. e.g.

{
CriticalSection cs;
...
Protected code
...
}

uses PRIMASK
