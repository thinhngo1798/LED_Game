10 /**
11  * Terminology (from USB Specification)
12  *
13  * Packet
14  * - A bundle of data organised in a group for transmission. Packets typically
15  * contain three elements: control information (e.g., source, destination, and
16  * length), the data to be transferred, and error detection and correction bits.
17  * 8 7 4 5
18  * +-----+---------+----------+------+
19  * | PID | Address | Endpoint | CRC5 | Token packets : IN, OUT, SETUP
20  * +-----+---------+----------+------+
21  * 8 11 5
22  * +-----+--------------------+------+
23  * | PID | Frame Number | CRC5 | Token packet : SOF
24  * +-----+--------------------+------+
25  * 8 0-8192 16
26  * +-----+----//-----+---------+
27  * | PID | Data | CRC16 | Data packets : DATA0, DATA1
28  * +-----+----//-----+---------+
29  * 8
30  * +-----+
31  * | PID | Handshake packet : ACK, NAK
32  * +-----+
33  *
34  *
35  * Phase - transaction has three phases.
36  * - Token packet
37  * - Data packet (optional)
38  * - Handshake packet (optional)
39  *
40  * Transaction
41  * - The delivery of service to an endpoint; consists of a token packet, optional data
42  * packet, and optional handshake packet. Specific packets are allowed/required
43  * based on the transaction type.
44  *
45  * Transfer
46  * - One or more bus transactions to move information between a software client
47  * and its function.
48  *
49  * Bulk Reads and Writes
50  * - Consist of a series of either OUT or IN transactions
51  * +--------+ +--------+ +--------+ // +--------+
52  * | OUT(0) | | OUT(1) | | OUT(0) | // |OUT(0/1)|
53  * +--------+ +--------+ +--------+ // +--------+
54  * +--------+ +--------+ +--------+ // +--------+
55  * | IN(0) | | IN(1) | | IN(0) | // | IN(0/1)|
56  * +--------+ +--------+ +--------+ // +--------+
57  *
58  * Control Setup Reads and Writes
59  * - Consists of 3 stages
60  * SETUP stage (1 SETUP transaction),
61  * DATA stage (0 or more IN/OUT transactions)
62  * STATUS stage (1 IN/OUT transaction opposite to that of DATA stage or IN if no DATA stage)
63  * +--------+ +--------+ +--------+ // +--------+ +--------+
64  * |SETUP(0)| | OUT(1) | | OUT(0) | // |OUT(0/1)| | IN(1) |
65  * +--------+ +--------+ +--------+ // +--------+ +--------+
66  * +--------+ +--------+ +--------+ // +--------+ +--------+
67  * |SETUP(0)| | IN(1) | | IN(0) | // | IN(0/1)| | OUT(1) |
68  * +--------+ +--------+ +--------+ // +--------+ +--------+
69  * +--------+ +--------+
70  * |SETUP(0)| | IN(1) |
71  * +--------+ +--------+
72  */
