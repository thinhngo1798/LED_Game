474  * Transfer Control Descriptor
475  * @verbatim
476  * +------------------------------+ Simple DMA mode (MLNO = Minor Loop Mapping Disabled)
477  * | Major Loop = | ==================================================
478  * | CITER x Minor Loop |
479  * | | Each DMA request triggers a minor-loop transfer sequence.
480  * | +--------------------------+ |<-DMA Req. The minor loops are counted in the major-loop.
481  * | | Minor Loop | |
482  * | | Each transfer | | The following are used during a minor loop:
483  * | | SADDR->DADDR | | - SADDR Source address
484  * | | SADDR += SOFF | | - SOFF Adjustment applied to SADDR after each transfer
485  * | | DADDR += DOFF | | - DADDR Destination address
486  * | | Total transfer is NBYTES | | - DOFF Adjustment applied to DADDR after each transfer
487  * | +--------------------------+ | - NBYTES Number of bytes to transfer
488  * | +--------------------------+ |<-DMA Req. - Attributes
489  * | | Minor Loop | | - ATTR_SSIZE, ATTR_DSIZE Source and destination transfer sizes
490  * |..............................| - ATTR_SMOD, ATTR_DMOD Modulo
491  * | | | |
492  * | +--------------------------+ | The number of reads and writes done will depend on NBYTES, SSIZE and DSIZE
493  * | +--------------------------+ |<-DMA Req. For example: NBYTES=12, SSIZE=16-bits, DSIZE=32-bits => 6 reads, 3 writes
494  * | | Minor Loop | | NBYTES must be an even multiple of SSIZE and DSIZE in bytes.
495  * | | Each transfer | |
496  * | | SADDR->DADDR | | The following are used by the major loop
497  * | | SADDR += SOFF | | - SLAST Adjustment applied to SADDR at the end of each major loop
498  * | | DADDR += DOFF | | - DLAST Adjustment applied to DADDR at the end of each major loop
499  * | | Total transfer is NBYTES | | - CITER Major loop counter - counts how many completed major loops
500  * | +--------------------------+ |
501  * | | SLAST and DLAST may be used to reset the addresses to the initial value or
502  * | At end of Major Loop | link to the next transfer.
503  * | SADDR += SLAST | The total transferred for the entire sequence is CITER x NBYTES.
504  * | DADDR += DLAST |
