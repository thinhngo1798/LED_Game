Detailed Description

template<typename T>
class USBDM::Nonvolatile< T >

Class to wrap a scalar variable allocated within the FlexRam area.

Size is limited to 1, 2 or 4 bytes.

Writing to the variable triggers an EEPROM update.
Ensures updates are completed before return.

Template Parameters
TScalar type for variable
Note
Instances should be placed in FlexRAM segment e.g.
__attribute__ ((section(".flexRAM")))
USBDM::Nonvolatile<char> a_nonvolatile_char;
Examples:
nonvolatile-example.cpp.
